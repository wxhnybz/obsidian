## 字符转askii码以及转回去
charCodeAt(i)
fromCharCode(i)
貌似：js的一些函数和属性使用小驼峰法进行命名
```
res+=String.fromCharCode(s.charCodeAt(i)-1)
```

```
这行代码的作用是将字符串`s`中位于索引`i`的字符的字符编码减去`1`，然后将这个新的字符编码转换回字符，并将这个字符追加到字符串`res`上。下面是这个过程的详细解释：
1. `s.charCodeAt(i)`: 获取字符串`s`中位于索引`i`的字符的字符编码。例如，如果`s`是`"abc"`且`i`是`1`，那么`s.charCodeAt(i)`的结果是`98`，因为`b`的字符编码是`98`。
2. `s.charCodeAt(i)-1`: 将上一步得到的字符编码减去`1`。继续上面的例子，这会得到`97`。
3. `String.fromCharCode(...)`: 将上一步得到的字符编码转换回字符。在我们的例子中，`97`对应的字符是`a`。
4. `res += ...`: 将上一步得到的字符追加到字符串`res`上。如果`res`原来是`"hello"`，现在它会变成`"helloa"`。
这种用法可以用于对字符串中的每个字符进行简单的加密或解密操作，通过改变字符编码来实现。
```
